## 30-min retro synthesis (M2 + M3 combined — full retro doc separate)

> Since M2 retro was missed during M2 close, fold both into this synthesis. Full version: `docs/retros/2026-05-13-m3-retro.md`.

### What worked

1. **3-agent parallel chats (MAIN orchestration + BE+1 + FE+1).** Day-15's 15-PR record relied on independent execution chains coordinating only at merge boundaries. No team-of-1 could have shipped that volume.
2. **Compression cadence.** M2 closed 2 days early; M3 closed 1 day late (net 1 day ahead of 4-week original baseline). M4's 4-day target only feasible because cadence is now muscle memory.
3. **Hardcoded-fallback pattern.** When Cloudflare Pages env injection failed (drift #2), hardcoded baseURL fallback in #122 unblocked production without waiting for upstream fix. Same pattern reused for #129 absolute callbackURL.
4. **Path C bridge for runtime config.** #115 LLMGateway DB-source bridge meant Composer + Curator could ship before F26 admin UI (deferred to M5) without losing operability.
5. **Repo public-flip for unlimited Actions.** Day-14 single-decision unblocked the entire M3 cascade. PM1 code is OSS-licensed already; private-by-default was costing more than it protected.

### What hurt

1. **5-fix BetterAuth chain (Day-15 → Day-17).** Each layer revealed the next. Caught on the user-side, not in pre-deploy smoke. Lesson banked.
2. **3-PR Cloudflare deploy chain.** First successful auto-deploy in 100+ runs landed Day-14. The Deploy workflow had been silently broken since project inception.
3. **Cascade-rebase storms.** Day-13 GHA outage left 9 PRs cascade-conflicting on resume. Day-14 saw repeat. Documented in `(ar)` pattern.
4. **Render Free SMTP block (Sept 2025 policy).** Caught by user complaining about magic-link emails not arriving; should have been caught during ADR-008 deploy-target verification.
5. **Quota / API rate limits.** Day-13 evening hit Anthropic API rate limits. Day-16 Yogesh's MAIN session hit personal quota. Day-17 entirely a refresh day.

### What to keep

1. **Visual-gate flag protocol** — `[VISUAL GATE PENDING]` in PR title is the canonical hold mechanism. Caught at least 4 design drifts in M3.
2. **3-agent parallel-chat coordination via paste-blocks** — no automation needed for this scale.
3. **Hard Rule 11 (ask before non-trivial decisions)** — every time I followed it, the answer surprised me in a useful way. Day-13 `(ae)` embedding-spec pause prevented a wrong unilateral edit.

### What to drop

1. **GitHub Actions free-tier scaffolding** — repo stays public through M5. Revisit at PM1 → PM2 transition only if pilot-data privacy requires private.
2. **5-min poll cadence during cascade drain** — 3-min worked just as well at Day-15 burst, with less context burn.
3. **Cross-agent paste-blocks for trivial rebases** — direct gh push from MAIN's worktree works when branches aren't worktree-locked. Reserve paste-blocks for worktree-locked or substantive review.
