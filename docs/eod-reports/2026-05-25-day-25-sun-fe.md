## Day-25 Sun lessons (no separate memory file tonight — Monday EOD or M5 retro will consolidate)

1. **Sandboxed string-diff for audit decisions saves 2hr+.** The F19 Option C decision (vs full rebuild) was made in <5 min using `/tmp/f19-diff-strings.mjs` — 118 OLD strings diffed against 282 NEW extract strings, surfaced only 2 false-positive divergences. Same pattern saved the F23 sanity check from a false-bundle-reject (initial raw 50% structural divergence flipped to actual 12.5% page-content ARIA).

2. **Bundle's structural top-level count can be misleading when canonical extract includes shell wrappers.** Always compare on the meaningful axis: page-CONTENT ARIA, not raw section counts. Future bundle sanity checks: compute BOTH (raw + page-content ARIA) and trust the latter.

3. **Day-23 stale-install lesson is the most predictable recurring blocker.** Every push after another agent has merged dep changes needs `pnpm install` first. Worth automating into a pre-push hook step (could check `pnpm-lock.yaml` mtime vs `node_modules/.modules.yaml` mtime).

Lesson recorded: when feedback says "still not identical" without direction, ASK before guessing the direction.
