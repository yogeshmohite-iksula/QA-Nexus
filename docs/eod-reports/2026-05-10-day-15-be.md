## Architectural wins worth highlighting

1. **Defense-in-depth CORS pattern** (PR #123 Layer 1 + PR #124 Layer 2) — Express middleware owns preflight + headers; BetterAuth `trustedOrigins` owns CSRF on the request body. Two layers, two policies, kept in sync via shared `AUTH_TRUSTED_ORIGINS` env var. Reusable for any future third-party-lib mount with browser-CORS gaps.
2. **Path C transitional bridge ADR pattern** (ADR-015) — when CLAUDE.md spec drifts from reality OR a UI is months out, ship a transitional bridge with explicit lifespan + retirement plan. Status `Transitional`, `Superseded by:` field pre-populated.
3. **3-iteration CI RCA discipline** (PR #115 fixes #1/#2/#3) — each failure was DIFFERENT surface; pull actual log via `gh run view --log-failed`, analyze with terse code in ctx_execute, file regression test for each fix.
