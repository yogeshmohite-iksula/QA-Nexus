Methods: `presignedUpload({contentType, filename, prefix?})` returns matched PUT + GET URLs (key naming `<prefix>/<YYYY-MM-DD>/<uuid>-<sanitized-filename>`), `presignedDownload({key})`, `listObjects({prefix?, limit?})`, `deleteObject(key)`, `health()` returns `{up,down,deferred}`. - **`apps/api/src/storage/storage.controller.ts`** — `POST /storage/presigned-upload` (Admin/Lead via `@Roles`, writes synchronous audit row per CLAUDE.md Rule 7) + `POST /storage/presigned-download` (any authed user, no audit per cost/value).
- **`apps/api/src/storage/zod-validation.pipe.ts`** — minimal NestJS pipe wrapping a Zod schema; throws BadRequest with formatted issues.
- **`apps/api/src/storage/r2.service.spec.ts`** — **14/14 jest unit tests pass** (mocked SDK). Covers deferred mode + configured mode + filename sanitization + TTL clamping + health up/down/deferred.
