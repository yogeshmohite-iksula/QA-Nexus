Connect to your Atlassian workspace

QA Nexus needs read + write access to fetch issues and sync test cases. You can scope access to specific projects in the next step.

What QA Nexus will access

  • Read issues, stories, epics, sprints, and custom fields from authorized Jira projects
  • Link QA Nexus test cases and defects to Jira issues, create defects as Jira Bugs
  • Post comments and status updates to linked issues (opt-in per project)
  • Never: delete issues, modify issue types, or change Jira project settings
You can revoke this access anytime from Integrations → Jira → Disconnect.
